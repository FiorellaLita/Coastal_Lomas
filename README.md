Title: Environmental and geographic predictors of seed plant diversity across South American terrestrial coastal lomas
Authors: Fiorella N. Gonzales, Holger Kreft, Luis Arapa, Dylan Cravenf

Data use guidelines and availability
Shapefiles/Lomas.SHP.zip: This file contains a shapefile with the polygons of coastal lomas across Peru and Chile.
Data/Metadata_Lomas.docx: This file provides the geographical location (latitude, longitude, country), area (ha and km2),
                         and observed plant species richness for 71 coastal lomas along Peru and Chile.
